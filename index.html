U5 — PROJECTS

Projects

Dual-Channel Function Generator with Wireless DSO

Portable test-and-measurement system built on an ESP32 microcontroller with Direct Digital Synthesis. Generates dual-channel output with adjustable frequency across sine, square, triangle, and arbitrary waveforms, with Normal and Sweep modes for frequency-response testing.

Paired with a wireless dual-channel Digital Storage Oscilloscope on a Raspberry Pi Pico for real-time waveform acquisition and monitoring.

8-Bit RISC Processor

Designed and implemented an 8-bit RISC processor for a Digital System Design course — ALU, registers, program counter, instruction decoder, and control unit, all built and verified from scratch.

Verified functionality through simulation and instruction execution testing, gaining hands-on experience in datapath design and control logic generation using Verilog HDL.

Li-Ion Battery Charger & Charge Indicator

Charging and monitoring system for a 3.7V lithium-ion battery using LM358 and LM324 op-amps, with automatic cut-off protection through voltage comparison to prevent overcharging.

Added a multi-level LED indicator that displays charge status using comparator-based voltage detection.

Hybrid EV Battery Cooling System

IoT-based thermal management system for EV lithium-ion battery packs in a 4S4P configuration, combining air and liquid cooling to improve safety, efficiency, and pack lifespan.

Custom battery holder with integrated liquid cooling channels, real-time temperature monitoring via ESP32, automatic cooling activation, and remote monitoring through an IoT dashboard.

Multi-Trainer Kit for Electronics Development

Integrated electronics development platform on an ATmega328P, combining an IC tester, component tester, single-channel DSO, dual-channel function generator, voltmeter, ammeter, and digital multimeter into one device.

A portable, cost-effective tool for electronics learning, circuit testing, troubleshooting, and prototyping.
